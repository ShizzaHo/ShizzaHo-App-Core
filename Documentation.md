# Документация по ShizzaHo App Core

## Начало.
**Как запустить это чудо инженерии?**

Запускаем *ShizzaHoAppCore.exe*, предварительно установив *NET 3.1*

**Создаем новый проект**

- Чтобы создать свой проект, нам необходимо зайди в компилятор и выбрать пукт **"Компилятор .ShizzaHoApp"** 

Далее мы выбираем пункт создания проекта, вводим все данные, и радуемся жизни!

- Далее переходим в папку созданного проекта и открываем скрипт по пути **Scripts/Start.cs** 

- Нас встретит пустой файл, в котором мы уже можем смело начинать -быдлокодить- тоесть, писать код.

**Пишем код по пальцам**

- Для начала нам стоит написать стартовый void, который ранер воспринимает как стартовую функцию.

```cs
void Start(){
// Сюда кодим
}
```

Start будет запускаться всегда первым, если же тебя не устривает такое, то делай вызов другого метода, например

```cs
void Start(){
    MyMegaVoid();
}

static void MyMegaVoid(){
    // Сюда кодим
}
```
